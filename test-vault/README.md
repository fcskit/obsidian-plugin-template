# Test Vault

This vault is for testing the Obsidian Plugin Template.

## Features to Test

- [ ] Plugin loads successfully
- [ ] Example modal opens
- [ ] Settings tab appears
- [ ] Debug messages show in dev build
- [ ] Hot-reload works (change code, auto-reloads)
- [ ] Tag filtering works
- [ ] Production build eliminates debug

## Testing Debug Features

Open DevTools (Ctrl+Shift+I / Cmd+Option+I) to see console output.

### Development Build
Debug messages should appear with component tags and optional custom tags.

### Production Build
Debug messages should NOT appear in the console.

## Hot-Reload

The hot-reload plugin is installed and configured. When you modify the plugin code and rebuild, Obsidian will automatically reload the plugin without manual intervention.

## Next Steps

1. Run `npm run dev` to build the plugin
2. Open this vault in Obsidian
3. Enable community plugins in Settings
4. Test the features above
